# 🌊 Dubrownik — Plan Podróży 6–10 maja

Interaktywny plan wycieczki do Dubrownika z mapą, zdjęciami i możliwością dodawania własnych lokacji.

## 🚀 GitHub Pages

1. Wrzuć zawartość repozytorium na GitHub
2. Wejdź w **Settings → Pages**
3. Wybierz **Source: Deploy from a branch** → `main` → `/ (root)`
4. Po chwili strona będzie dostępna pod adresem `https://<twój-nick>.github.io/<nazwa-repo>/`

## 📱 Funkcje

- **Plan dnia** — 5 dni, każdy zwijany, z godzinami i wskazówkami
- **Mapa interaktywna** — Leaflet OSM, kolorowe pinezki per dzień, popup ze zdjęciem
- **Dodaj lokację** — własne miejsca z koordynatami zapisywane w `localStorage`
- **Mobile-first** — wygodne na telefonie

## ✈️ Praktyczne info

- **Lotnisko → centrum:** bus nr 11 (~30 min, ~7 €) lub taxi (~30 min, ~35 €)
- **Dubrovnik Pass:** warto kupić online — mury (40 €), muzea, busy gratis
- **Domestico (śniadanie 10 maja):** Močići 24A, Čilipi — 3 min od lotniska, otwarte od 8:00
