# 📋 INSTRUKCJA UŻYCIA

## 🎯 Masz 3 Opcje:

### 1️⃣ GitHub Pages (POLECANE - Darmowy Hosting!)
- Wgraj na GitHub → Włącz GitHub Pages
- Otrzymasz link typu: `https://twojanazwa.github.io/dubrovnik-trip/`
- Pełna instrukcja w pliku `DEPLOYMENT.md`

### 2️⃣ Netlify Drop (NAJSZYBSZE - 30 sekund!)
- Wejdź na https://app.netlify.com/drop
- Przeciągnij cały folder
- Gotowe! Otrzymasz link typu: `https://random-123.netlify.app`

### 3️⃣ Lokalnie (Tylko Dla Ciebie)
- Rozpakuj folder
- Kliknij dwukrotnie `index.html`
- Otwiera się w przeglądarce!

## ✅ Szybkie Kroki (GitHub):

1. Stwórz konto na GitHub.com
2. New Repository → Nazwa: `dubrovnik-trip`
3. Upload files → Przeciągnij WSZYSTKIE pliki
4. Settings → Pages → Source: main, / (root) → Save
5. Poczekaj 2 minuty
6. Twoja strona działa! 🎉

## 🔧 Jak Zaktualizować Plan?

### Zmienić Miejsca:
1. Otwórz `index.html` w edytorze tekstu (Notepad++, VS Code, Sublime)
2. Znajdź: `const locations = [`
3. Edytuj miejsca, daty, opisy
4. Zapisz i wgraj ponownie na GitHub

### Zmienić Kolory:
1. Znajdź: `:root {` (na początku pliku)
2. Zmień wartości kolorów
3. Zapisz

## ❓ FAQ

**Q: Czy muszę mieć GitHub?**  
A: Nie! Możesz użyć Netlify Drop lub otworzyć lokalnie. Ale GitHub daje Ci stały, darmowy link.

**Q: Czy to zadziała offline?**  
A: Tak! Po pierwszym załadowaniu mapa i wszystko działa bez internetu.

**Q: Jak dodać własne miejsca?**  
A: Otwórz `index.html`, znajdź `const locations`, dodaj nowy wpis z nazwą, współrzędnymi (z Google Maps) i opisem.

**Q: Mogę to użyć dla innej wycieczki?**  
A: TAK! Po prostu zmień miejsca, daty, zdjęcia w pliku HTML.

**Q: Ile to kosztuje?**  
A: 0 złotych. Wszystko darmowe. Zawsze.
